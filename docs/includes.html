Including Other Peri.dot Files

If there is another peridot file you want to include you can use the include keyword.

var testmodule = include('testmodule') # The locations that the above will check in are listed in __peridot__.path.

Including Built-In Modules

There are a couple of built-in modules as well:

  • sys
  • file
  • datetime
  • periscope

Sys

var sys = include('sys') print(sys.args) # Gets the command line arguments. # Ex: # peridot test.peri dosomething # Returns ['test.peri', 'dosomething']. print(sys.terminalsize()) # Returns a tuple containing the terminal size.

File

var file = include('file') # Modes: # r : Open file for reading # rb : Open file in binary for reading # r+ : Open file for reading and writing # w : Open file for writing # wb : Open file in binary for writing # w+ : Open file for writing and reading # wb+ : Open file in binary for writing and reading # a : Open file for appending # ab : Open file in binary for appending # a+ : Open file for appending and reading # ab+ : Open file in binary for appending and reading var f = open('test.txt') # Mode defaults to r print(f.read()) # Returns the contents of test.txt. f.close() var f = open('test.txt', mode='w') f.write('Hello Peri.dot!') # Returns the contents of test.txt. f.close()

Datetime

var datetime = include('datetime') var d = datetime.now() print(d) # Gets the current time var d = datetime.datetime( y=2020 # Year m=8 # Month d=31 # Date H=11 # Hour M=57 # Minute S=26 # Second f=192873 # Microsecond ) print(d.strftime('%c')) # Create datetime object with custom time. # Then, print with strftime format '%c'
