Booleans

Booleans are values that are either True or False. These are used in if, switch, and while statements.

The bool type

bool(1) # True bool('') # False

If Statements

If statements are used when you want to execute separate code depending on certain conditions.

var x = 3 if (x == 1) { print('x is 1') } elif (x == 2) { print('x is 2') } elif (x == 3) { print('x is 3') } else { print('x is not 1 or 2 or 3') }

Switch Statements

Switch statements are another way of doing if statements.

var a = 3 switch (var x as a) { # ^^^^^^^^^^ # NOTE: If the var keyword is not there, x must already # be initialized as the same type as a. when (x == 1) { # ^^^^^^ # NOTE: You can not access any variable except for the specified # variable at the start of the switch statement. print('x is 1') } when (x == 2) { print('x is 2') } when (x == 3) { print('x is 3') } else { print('x is not 1 or 2 or 3') } }

For Looops

For loops iterate through a list.

var a = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h'] for (var i in a) { #^^^^^^^^^^ #NOTE: If the var keyword is not here, i must already be # initialized as the same type as all of the contents of a. print(i) }

They can also repeat actions several times

var a = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h'] for (var i in range(10)) { print(str(i)) }

While Looops

While loops execute code repeatedly until the given condition is no longer True.

var x = 100 while (x >= 1) { x = x - 1 }

Break and Continue

break() and continue() are used in loops to "move around".

break() will break out of the loop.

for (var i in range(10)) { if (i >= 5) { break() # The loop will break at iteration 5. } print(str(i)) # 0~4 will be printed. 5+ will not. }

continue() will continue to the next iteration of the loop.

var i = 0 while (True) { if (i == 5) { continue() # The loop will skip the rest of iteration 5. } print(str(i)) # Will keep printing all numbers except 5 }
