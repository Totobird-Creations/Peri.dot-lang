Pre-Initializing Variables

Variables containing Null can be set to anything else. Pre-initializing will set the variable to Null without you typing it all out.

var test print(str(test)) # This will print Null

Initializing Variables

To initialize a variable, use the var keyword.

var test = 'hello' print(str(test))

Assigning Variables

Assigning a variable is similar to initializing it. However, the variable type can not be changed.

var test = 'hello' test = 'Hello World!' # This will assign 'Hello World!' to test. test = 10 # This will throw a TypeError. var test = 10 # You can reinitialize the variable to change its type. var test = Null # Null variables can be set to anything else. test = [1, 2, 3, 4] # This works
