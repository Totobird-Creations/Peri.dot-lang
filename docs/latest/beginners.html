Hello World

Just like every other documentation for a programming language, let's start with a hello word program.

The print built-in function

The print built-in function takes a strings and prints it out to the console.

print('Hello ' + 'World!')

Strings are surrounded by quotes or double quotes.

Integers and Floating Point Numbers

You need numbers in a programming language. There's no more explanation than that.

The str type

The str type takes in an object and converts it into a string. This is especially useful when you want to print non string objects.

print(str( 10 + 10 * 2 - 8 / 4 ))

Note that due to this programming language being strongly typed, most situations will not allow you to use a different type than expected.
For example:

print(str( 10 + 10.0 ))

The above script will throw a type error saying that you can not add a float to an int. Also,

print(str( 10 / 3 ))

The above script will print 3 instead of 3.333

Arrays and Tuples

Arrays arean object that can contain many objects. In Peri.dot, all of the contents must be the same type.

print(str( [1, 2, 3, 4] ))

Tuples are similar to arrays. However they can hold any type.

print(str( (1, 2, 3, 4) ))
