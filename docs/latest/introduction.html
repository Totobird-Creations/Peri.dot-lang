Introduction

Peri.dot is a strongly typed interpreted language, with type inference, implemented in Python. The file extension is ".peri"

Installation

Peridot can be installed using pip. The following modules are also required:

  • click 7.1.2
  • colorama 0.4.3
  • xdgappdirs ^1.4.5
  • pygments ^2.6.1
  • prompt_toolkit ^3.0.7
  • perimod ^1.0.1

If you use pip, they should install automatically.

pip install peridot

Usage

# Executing a file: peridot file.peri # Opening the repl: peridot # Help and other commands: peridot -h
