Basic Function

Functions are groups of code that can be called multiple times. Functions MUST return a value using the return keyword. The return type must also be specified.

var helloword = func() -> Null { print('Hello World!') return(Null) } helloword() # This calls the function

Function Arguments

The type of arguments in Peri.dot functions must be specified. If the argument type is Null, the argument can take in any type.

var greet = func(name: str) -> Null { print('Hello, ' + name + '!') return(Null) } greet('Peri.dot')

Function Default Arguments

Default arguments are arguments that have a default. The argument type comes from the default value. Default arguments that are Null can take in any value.

var greet = func(name: str, emph=1) -> Null { print('Hello, ' + name + '!'.repeat(emph)) return(Null) } greet('Peri.dot') # emph will default to 1. greet('Peri.dot', emph=10) # emph will be set to 10.

Returning

The return statement must be used in every function to return a value. The return value type is the number after the ->.

var add = func(a: int, b: int) -> int { return(a + b) } print( add(18, 32) ) # Should print 50.
